## ⚠️ LIMITS & RULES

```
┌─────────────────────────────────────────────────────────────┐
│                    ATURAN PENTING!                          │
└─────────────────────────────────────────────────────────────┘

1. 🚫 VIEW QUOTE: Max 3x/hari
   View ke-1, 2, 3  →  +1 XP each  ✅
   View ke-4, 5, 6  →  +0 XP       ❌
   Reset besok jam 00:00

2. 🚫 SELF-LIKE: Tidak dapat +5 XP
   Like quote sendiri  →  +3 XP (like)  ✅
                       →  +0 XP (receive) ❌

3. ✅ NO LIMIT untuk:
   • Buat quote (unlimited)
   • Like quote orang (unlimited)
   • Comment (unlimited)
   • Share (unlimited)

4. 📊 SEMUA TERCATAT
   Semua aktivitas XP masuk database
   Admin bisa monitor aktivitas mencurigakan
```
